Hosted on GitHub Pages — embed using Notion's `/embed` block.

## 📁 Files

- `widget-flip-clock.html` — 4-tile flip clock (one tile per digit), with colour customisation and date display
- `widget-compact-flip-clock.html` — 2-tile flip clock (hours block + minutes block)
- `widget-pomodoro.html` — Pomodoro timer with focus/break cycles, session history, and ring progress display
- `widget-music-player.html` — Embedded Spotify or YouTube Music player with compact/full view toggle
